**What the script does:**
1. Iterates through archetypeVoices array (17 total entries including 3 Roaster files that may already exist)

- 7 new Lovebomber files generated (~230-350KB each)

- Script completes with "✅ All archetype reveal files generated!"

**Error handling:**
- If API key invalid or missing: Script exits with error
