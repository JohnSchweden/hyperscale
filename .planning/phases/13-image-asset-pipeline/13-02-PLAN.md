<task type="auto">
  <name>Task 1: Create image generation pipeline script</name>
  <files>scripts/generate-images.ts, package.json</files>
  <action>
    Install sharp as a dev dependency:
    ```bash
    bun add -D sharp @types/sharp
    ```

    Create `scripts/generate-images.ts` following the existing `scripts/generate-voice.ts` pattern.

    The script should:

    1. **Read GEMINI_API_KEY from env** — exit with clear error if missing
    2. **Import IMAGE_PROMPTS** from `./prompts/image-prompts`
    3. **Accept CLI args** for filtering:
       - `bun scripts/generate-images.ts` — generate all
       - `bun scripts/generate-images.ts bankrupt pragmatist` — generate only those IDs
       - `bun scripts/generate-images.ts --model gemini-3.1-flash-image-preview` — switch model
       - `bun scripts/generate-images.ts --dry-run` — print prompts without generating
    4. **Category-to-directory mapping:**
       ```typescript
       const CATEGORY_DIR: Record<string, string> = {
         incident: "public/images/incidents",
         outcome: "public/images/outcomes",
         archetype: "public/images/archetypes",
         death: "public/images/deaths",
       };
       ```
    5. **Generate each image:**
       - Use `ai.models.generateContent` with `responseModalities: ["IMAGE"]`
       - Model default: `gemini-2.5-flash-image` (overridable via --model flag)
       - Set safety settings to `BLOCK_ONLY_HIGH` for all 4 categories
       - Extract base64 image data from response
    6. **Convert PNG to WebP via sharp:**
       - Resize: max 1024px width (withoutEnlargement: true)
       - WebP quality: 85
       - Log output file size
    7. **Save to correct directory:** `{CATEGORY_DIR[category]}/{id}.webp`
    8. **Rate limiting:** 7-second delay between requests (stays under 10 IPM Tier 1 limit)
    9. **Error handling:**
       - On safety filter block (finishReason: "SAFETY"): log warning, continue to next prompt
       - On 429 rate limit: wait 60 seconds, retry once
       - On other error: log, continue to next prompt
       - At end: print summary (generated, skipped, failed)
    10. **Skip existing:** If output file already exists and --force not passed, skip it (allows incremental regeneration)

    Add a `generate:images` script to package.json:
    ```json
    "generate:images": "tsx scripts/generate-images.ts"
    ```
