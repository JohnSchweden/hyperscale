## Quick Reference: Format Decision Tree

```
Incident/Outcome Type
│
├─ DEATH ENDING → Use death_to_template mapping
│
├─ ARCHETYPE REVEAL → Use archetype_to_template mapping
│
├─ FEEDBACK OVERLAY
│   │
│   ├─ Pattern = escalating-disaster / escalation → GIF if available
│   │
│   ├─ Pattern matches incident_mapping → Static meme
│   │
│   └─ No match → Fall back to Gemini AI
│
└─ LINKEDIN SHARE → Always static meme (GIFs render as thumbnails)
```

## Adding to Template Database

To add these to `data/templates/gif-templates.json`:

1. Add new `death_mapping` section with death types → template IDs
2. Add new `archetype_mapping` section with archetype → template IDs
3. Mark templates as `gif: true` for animated versions
4. Prioritize templates in `best_for` array
