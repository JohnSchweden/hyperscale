   - Call backend API endpoint (/api/gemini/live)
   - Receive streaming audio chunks via ReadableStream
   - For each chunk:
     a. Decode base64 to Uint8Array
     b. Convert Int16 to Float32 (divide by 32768.0)
     c. The AudioWorklet handles 24kHz→48kHz conversion internally
     d. Send to AudioWorklet via postMessage

6. Handle cleanup on unmount
7. Return { startSession, stopSession, isPlaying, error }

Key details:
- AudioContext must be created with { sampleRate: 48000 } for browser playback
- Base64 decode: atob() then Uint8Array
- Int16→Float32: sample / 32768.0
- The worklet does resampling - send 24kHz data as-is
- Send to worklet: workletNode.port.postMessage({ type: 'chunk', data: float32Array })

Note: For initial implementation, you may need to create a simple API route at src/app/api/gemini/live/route.ts that calls the geminiLiveProxy service.
